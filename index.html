Minha Colinha de SQL Server

Guia de consulta rápida para o dia a dia

⚙️ Configurações e Atalhos

  • Nome do Servidor: No CMD, digite hostname.
  • Nova Consulta: Ctrl + N.
  • Comentários: Use -- para uma linha ou /* texto */ para blocos.
  • Salvar: Arquivo > Salvar (Sempre conecte ao banco após reabrir).

💾 Como Importar Banco (.bak)

  1. Botão direito em Databases > Restore Database.
  2. Selecione Device > Clique nos ... > Add.
  3. Escolha o arquivo .bak e dê OK até concluir.
  4. Atualize a pasta para ver as tabelas em Tables.

🔍 Consultas de Seleção (DQL)

O SELECT define quais colunas retornar, e o FROM qual tabela.

-- Selecionar tudo (boa prática: pular linhas)

SELECT * FROM DimProduct;

-- Selecionar colunas específicas e valores únicos

SELECT DISTINCT ColorName, UnitPrice 
FROM DimProduct;

-- Limitar linhas (Top e Percent)

SELECT TOP (100) * FROM DimProduct;
SELECT TOP (10) PERCENT * FROM DimProduct;

🚀 Próximos Passos (Essenciais)

Estes são os comandos que completam sua base inicial:

-- WHERE: Filtrar dados

SELECT * FROM DimProduct 
WHERE ColorName = 'Blue' AND UnitPrice > 100;

-- ORDER BY: Ordenar (ASC ou DESC)

SELECT * FROM DimProduct 
ORDER BY UnitPrice DESC;

-- Funções de Agregação

SELECT 
    COUNT(*) AS TotalProdutos,
    AVG(UnitPrice) AS PrecoMedio,
    SUM(UnitPrice) AS SomaTotal
FROM DimProduct;
