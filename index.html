Guia de Consulta SQL

Fundamentos, Filtros, Agregações e Agrupamentos

1. Fundamentos e Importação

Para que serve: O SQL (Structured Query Language) é a linguagem padrão para interagir com bancos de dados relacionais.

Importação no SQL Server (SSMS)

  • Botão direito em Databases > Restore Database.
  • Selecione Device e localize o arquivo .bak.
  • Acompanhe o progresso até a mensagem de sucesso.

Conexão e Visualização

Para descobrir o nome do servidor, digite hostname no CMD. Para abrir uma nova aba de comandos: Ctrl + N.

2. Seleção de Dados (SELECT)

SELECT: Define quais colunas retornar. FROM: Define a tabela de origem.

-- Selecionar todas as colunas SELECT * FROM DimProduct; -- Selecionar colunas específicas (Melhor Prática) SELECT ProductName, BrandName, UnitPrice FROM DimProduct;

DISTINCT, TOP e AS

  • DISTINCT: Remove duplicatas, trazendo valores únicos.
  • TOP (n): Limita a quantidade de linhas retornadas.
  • AS (Alias): Renomeia temporariamente uma coluna ou tabela.
-- Selecionar os 10% produtos com preços distintos e renomear coluna SELECT TOP 10 PERCENT DISTINCT ColorName AS Cores_Disponiveis FROM DimProduct;

3. Filtros e Ordenação (WHERE & ORDER BY)

WHERE: Filtra as linhas com base em condições. ORDER BY: Ordena o resultado (ASC ou DESC).

-- Filtro com números, textos e ordenação SELECT ProductName, UnitPrice FROM DimProduct WHERE UnitPrice >= 1000 AND ColorName = 'Black' ORDER BY UnitPrice DESC; -- Filtro de Datas (Formato 'AAAA-MM-DD') SELECT FirstName, BirthDate FROM DimCustomer WHERE BirthDate > '1970-12-31';

Operadores Especiais

  • IN: Verifica se um valor está em uma lista.
  • BETWEEN: Filtra um intervalo (inclusive).
  • LIKE: Busca padrões (% representa qualquer texto).
  • IS NULL / IS NOT NULL: Verifica campos vazios.
-- Exemplo LIKE e BETWEEN SELECT * FROM DimProduct WHERE ProductName LIKE '%Home%' AND UnitPrice BETWEEN 50 AND 100; -- Exemplo IN e IS NULL SELECT * FROM DimStore WHERE Status IN ('On', 'Pending') AND CloseDate IS NULL;

4. Funções de Agregação

Servem para realizar cálculos sobre um conjunto de valores e retornar um único valor.

Função Descrição
COUNT() Conta o número de linhas ou valores não nulos.
SUM() Soma os valores de uma coluna numérica.
AVG() Calcula a média aritmética.
MAX() / MIN() Retorna o valor máximo ou mínimo.
-- Resumo de vendas SELECT COUNT(*) AS TotalVendas, SUM(SalesQuantity) AS QtdTotal, AVG(UnitPrice) AS PrecoMedio, MAX(UnitPrice) AS ProdutoMaisCaro FROM FactSales;

5. Agrupamentos (GROUP BY & HAVING)

GROUP BY: Agrupa linhas que têm os mesmos valores em colunas específicas para aplicar funções de agregação.

HAVING: É o "WHERE" dos grupos. Filtra o resultado após o agrupamento.

-- Quantidade de funcionários por gênero (Apenas grupos com mais de 100) SELECT Gender, COUNT(*) AS Total FROM DimEmployee WHERE DepartmentName = 'Finance' -- Filtra linhas antes de agrupar GROUP BY Gender HAVING COUNT(*) > 100; -- Filtra o resultado do grupo
Diferença Crucial: O WHERE filtra dados antes do cálculo. O HAVING filtra os resultados já calculados pelo GROUP BY.
